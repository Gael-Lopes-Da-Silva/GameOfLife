### ‚ùì How to use

- C - Clear all cells
- R - Randomize all cells
- P - Open the patterns menu
  - 1 - Create a block
  - 2 - Create a blinker
  - 3 - Create a glider
  - 4 - Create a pentomino

### ‚ùì How to compile

Download the go compiler [here](https://go.dev/dl/).

Then run this command:
~~~shell
$ go -ldflags -H=windowsgui build .
~~~

Or this one:
~~~shell
$ bash build.sh
~~~
